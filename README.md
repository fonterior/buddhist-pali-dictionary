# Буддийский словарь палийской традиции / Buddhist Pali Dictionary

Справочник терминов палийского буддизма на основе сутт Палийского канона.

## О проекте

Словарь содержит более 200 терминов с подробным описанием:
- Этимология (корни и словообразование)
- Основное значение
- Контексты употребления
- Ключевые цитаты из сутт
- Связанные термины (со ссылками)
- Стадия пути (sīla / samādhi / paññā)
- Значение для практики

## Структура

| Файл | Описание |
|------|----------|
| `glossary.md` | Полный словарь со ссылками |
| `glossary.txt` | Полный словарь |

## Разделы словаря

1. Базовые категории (№1–30)
2. Пять совокупностей (№31–39)
3. Пути и практики (№40–69)
4. Помехи и загрязнения (№70–79)
5. Зависимое возникновение (№80–90)
6. Плоды пути (№91–98)
7. Факторы ума и способности (№99–124)
8. Социальные и этические категории (№125–131)
9. Умственные факторы (№132–143)
10. Время, распад и процесс (№144–151)
11. Общие категории опыта (№152–160)
12. Учение и передача (№161–201)

## Лицензия

Только для некоммерческого использования, для образовательных целей.
